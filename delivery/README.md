## Как редактировать контент

### Цены, тексты, блоки — через PageBuilder

1. Войдите в `/manager/`
2. Откройте нужный **Ресурс** (страницу) из дерева слева
3. Вкладка **«Конструктор»** — редактируйте блоки визуально:
   - Каждый блок (герой, прайс-таблица, калькулятор) раскрывается кликом
   - Цены меняются в полях блока, сохраняйте кнопкой «Сохранить»

### Телефон, адрес, email — через Настройки

`Управление → Настройки сайта (ClientSettings)` — первые строки: телефон, адрес, соцсети.

### SEO-поля

Вкладка **«SEO»** на странице ресурса: title, description, canonical.

## Технический стек

- **CMS**: Evolution CMS 3 (evocms-community/evolution)
- **PHP**: 8.3-FPM
- **БД**: MariaDB (база `kosmos`)
- **Веб-сервер**: Nginx
- **Шаблоны**: Blade
- **Фронтенд**: Vanilla JS, CSS Custom Properties (без фреймворков)
- **Шрифты**: Unbounded + Manrope (Google Fonts)

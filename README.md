# Космос Детейлинг Студия — сайт (kosmos-detail.ru)

Статический сайт детейлинг-студии на Васильевском острове (СПб, ул. Кораблестроителей, 14).
Без бэкенда и сборки — открывается в любом браузере, заливается на любой хостинг.

## Как запустить локально
```bash
cd сайт
python3 -m http.server 8000
# открыть http://localhost:8000
```
Просто открыть `index.html` двойным кликом тоже работает (но карта/шрифты лучше грузятся через http).

## Структура
```
index.html              — главная (витрина, пакеты, работы, FAQ, форма)
zashchita-kuzova.html   — кластер: PPF / винил / антихром / мото / брендирование
polirovka-keramika.html — кластер: полировка / керамика / жидкое стекло / антидождь / диски
tonirovka-optika.html   — кластер: тонировка / бронирование и шлифовка фар / лобовое
detailing-salona.html   — кластер: химчистка / кожа / потолок / перетяжка / озонация
komfort.html            — кластер: шумоизоляция / доводчики / сигнализация / PDR / предпродажная
kontakty.html           — контакты, карта Яндекс, форма записи
assets/css/styles.css   — дизайн-система (космическая премиум-эстетика)
assets/js/main.js       — анимации, лайтбокс, форма, меню, FAQ
assets/img/             — фото работ (web, ≤1920px) + thumb/ (≤800px)
assets/photos/          — исходные оригиналы (не публикуются, см. robots.txt)
sitemap.xml, robots.txt — для поисковых систем
semantic-core.md        — семантическое ядро и SEO-структура по кластерам
build_pages.py          — генератор кластерных страниц (для повторной правки контента)
```

## Форма заявки
Сейчас форма показывает сообщение «Заявка отправлена» без реальной отправки (демо).
Для приёма заявок подключите один из вариантов:
- отправку на e-mail/Telegram через сервис (например, Formspree, Getform) — заменить `<form data-lead>`;
- или CRM/обработчик на вашем хостинге.

## Подключить перед запуском
- Яндекс.Метрику и Google Analytics (вставить счётчики перед `</head>`).
- Яндекс.Вебмастер + Google Search Console (добавить сайт, отправить sitemap.xml).
- Карточки в **Яндекс.Картах** и **2ГИС** — главный источник локального трафика (запустить параллельно).

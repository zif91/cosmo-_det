# Семантическое ядро и SEO-структура — Космос Детейлинг Студия

> Регион: **Санкт-Петербург, Васильевский остров** (Кораблестроителей 14, Намыв).
> Частотности (Wordstat) **указаны ориентировочно** — у инструмента xmlriver нет эндпоинта Wordstat,
> поэтому объёмы здесь это экспертная оценка по интенту, а не выгрузка. Перед рекламой свериться в
> Яндекс.Wordstat (wordstat.yandex.ru) с привязкой к региону «Санкт-Петербург».

## Принцип кластеризации

Каждый **коммерческий кластер = отдельная посадочная страница**. Внутри страницы — H1 под
главный интент, H2/H3 под подзапросы и услуги. Запросы сгруппированы по **интенту**
(коммерческий / информационный / гео / брендовый), а не по форме слова.

## Технические рекомендации
- Уникальные `<title>` / `<meta description>` / `<h1>` на каждой странице (реализовано).
- Schema.org `AutoRepair` + `LocalBusiness` (address, geo, openingHours, telephone) на всех страницах.
- ЧПУ-имена файлов по кластеру, перелинковка главная ↔ кластеры ↔ контакты.
- `sitemap.xml`, `robots.txt` (реализовано). Подключить Яндекс.Вебмастер и Метрику.
- Параллельно с сайтом — карточки в Яндекс.Картах и 2ГИС (главный источник локального трафика).
